# Photos & video go in this folder

Drop the files in here using these **exact names** and the site picks them up
automatically. Until a file exists, the site shows a neat labelled placeholder
in its place — so the site already looks finished and just gets better as you
add real photos.

| File name to use            | What it should be                                  |
| --------------------------- | -------------------------------------------------- |
| `hero.jpg`                  | The big background photo — a strong, warm portrait of an elder |
| `jungle-path.jpg`           | The bush / jungle path they have to walk to        |
| `pessila-washroom.jpg`      | Pessila's metal-sheet washroom                     |
| `pessila.jpg`               | A photo of Pessila herself                         |
| `jennifer.jpg`              | Jennifer with the yellow pots                      |
| `elder-kitten.jpg`          | The elder with the kitten (used as the video cover too) |
| `kitten-video.mp4`          | The video with the kitten                          |
| `companionship.jpg`         | A visiting / caring photo                          |
| `elder-1.jpg`               | Any elder in the community                          |
| `elder-2.jpg`               | Any elder in the community                          |
| `elder-3.jpg`               | Any elder in the community                          |

## Tips
- `.jpg`, `.jpeg`, `.png` all work for photos — but the name (before the dot)
  must match the table. If your photo is a PNG, either rename it to `.jpg`
  anyway won't work — instead change the filename in `index.html`, or just
  re-save it as a real `.jpg`. Easiest is to keep the names above.
- Keep photos reasonably sized (under ~2 MB each) so the site loads fast.
- For privacy/safety: only use photos you have permission to share publicly.
